# Code of Conduct

## Our standards

Participants are expected to:

- Be respectful and constructive
- Focus feedback on ideas and implementation details
- Assume good intent and ask clarifying questions
- Keep discussions relevant to the project

Unacceptable behavior includes harassment, personal attacks, or discriminatory language.

## Enforcement

Project maintainers may remove comments, issues, or pull requests that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact:

- linkedin.com/in/aneekhait/

All reports will be reviewed and handled in good faith.
