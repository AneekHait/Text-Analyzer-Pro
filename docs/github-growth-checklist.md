# GitHub Growth Checklist

Use this checklist in the GitHub web UI to improve discoverability.

## Repository settings

- Set repository description to include:
  - text clustering
  - excel
  - python
  - GUI
  - TF-IDF
- Add topics:
  - `text-clustering`
  - `nlp`
  - `excel`
  - `python`
  - `tkinter`
  - `kmeans`
  - `dbscan`
  - `agglomerative-clustering`
  - `tf-idf`
  - `keyword-extraction`
  - `pca`
  - `tsne`
- Enable Discussions (optional but useful)
- Pin this repository on your GitHub profile

## Publishing rhythm

- Ship at least one release per month
- Keep `CHANGELOG.md` updated for each release
- Respond to issues quickly
- Label first-timer tasks with `good first issue`

## Distribution

- Post release notes on LinkedIn and X
- Share demos in relevant Python/NLP communities
- Add screenshots or a short GIF to README
