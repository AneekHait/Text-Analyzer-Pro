# Fund Feature Requests & v2.0 — Text Analyzer Pro

Thank you for considering support. Contributions fund prioritized feature development, maintenance, hosting, and UX improvements for Text Analyzer Pro (v1.5) and the upcoming v2.0.

## Why fund?
- Accelerate delivery of requested features (visualizations, batch processing, CLI improvements).
- Pay for testing, CI, and cross-platform packaging.
- Enable priority support and early-access betas.

## How funds are used
- Development & implementation of funded feature requests
- Automated testing, packaging, and distribution
- Cloud/hosting costs for demo services and CI
- Tooling and design improvements

## Requesting a funded feature
1. Open a GitHub issue titled: `[Feature Request] Short title` and include a clear description, example data, and the expected outcome.
2. If you want priority handling, add a comment linking a donation reference or become a sponsor.
3. The maintainer will respond with an estimate, timeline, and whether the request will be scheduled for v2.0 or an earlier patch.

## Priority & perks (example)
- Supporter (€5–€10 / one-time): Public thank-you in release notes.
- Sponsor (€10+/month): Early access to beta builds + priority in scheduling.
- Patron (€50+/month): Feature co-design call + priority implementation + mention in About dialog.

## Transparency
- Funded features and how funds were spent will be listed in release notes and the repo CHANGELOG.
- Major donors (opt-in) will be credited in the About dialog and release notes.
